# cell_formation_problem

### Общая информация

> В данном проекте предложено решение NP-задачи Cell Formation Problem.

### Более подробное описание проблемы

> Cell Formation Problem - Есть определнное кол-во машин и определённое кол-во деталий. У каждой детали есть перечень машин, на которых она должны быть обработана. У каждой машины также присутствует схожий перечень. Нужно максимально эффективно произвести кластеризацию машин и деталей. Идеальная ситуация - когда 1 деталь требует только обработки 1 машины и 1 машины спецциально построена для этой детали.

### Ограничения

> У каждой машины должны быть как минимум одна  деталь для обработки и у каждой детали должна быть как минимум 1 машина.

### Структура проекта

`main.go`- файл с основной логикой</br></br>
`details/details.go` - содержит основные методы для работы c деталями</br></br>
`claster/cluster.go` - содержит основные методы для работы с кластеризацией</br></br>

`data/data.txt` - сюда помещаются входные данный (вывод ответа происходит в консоль)</br></br>
`data_inputs` - папка содержит входные данные</br></br>
